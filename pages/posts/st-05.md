### 进阶啦--超级高效的命令

find：
```

`find` 递归搜索文件

`find / -name "xxx"` 搜索xxx文件

`find / -name "*.txt"` 搜索所有txt文件

`find / -size +10M` 搜索大于10M的文件
```

grep：
```

`grep` 全局搜索内容

`grep "xxx" file` 搜索file文件中包含xxx的内容

`grep -n "xxx" file` 搜索file文件中包含xxx的内容，并显示行号

`grep -v "xxx" file` 搜索file文件中不包含xxx的内容
```

管道符：
```
`|` 管道符，将前一个命令的输出作为后一个命令的输入

`|` 常用于命令组合，如：

`ls -l | grep .txt` 列出当前目录下所有txt文件

`ps -ef | grep nginx` 查找nginx进程
```

echo：
```

`echo` 打印内容到屏幕

`echo hello` 打印hello

`echo $PATH` 查看环境变量

`echo $HOME` 查看家目录
```

vim：
```

`vim` 编辑器

`vim file` 打开文件

`i` 进入编辑模式

`esc` 退出编辑模式

`:wq` 保存并退出

`:q!` 强制退出不保存
```
